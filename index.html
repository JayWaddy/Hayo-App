About Hayo

Hayo is a project made to have both a Planco translator and dictionary all together in the same application. Thought there are already Planco translators on the web, I thought it would be fun to give it a shot and not only show that I am capable of creating an application in JavaScript, but also create a layout that still falls under the Planet Coaster style as a redesign.

This project is in no way affiliated with Planet Coaster or Frontier Development plc. This app was simply created as a test of strength and portfolio entry.

Feel free to send any questions or feedback to info@hayo.com that pertain to the website, NOT to Planet Coaster or Frontier Development plc. Any inquiries about the Planco language should go directly to Frontier Development plc.
